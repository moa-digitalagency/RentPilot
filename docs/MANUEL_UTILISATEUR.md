# Manuel Utilisateur

Ce guide explique comment utiliser les fonctionnalités de RentPilot pour chaque type d'utilisateur.

## 1. Connexion et Premiers Pas

Pour accéder à votre espace, rendez-vous sur la page de connexion, entrez votre email et votre mot de passe.
*   **Mot de passe oublié ?** Contactez votre administrateur.
*   **Première connexion** : Si vous êtes locataire, vérifiez votre email pour le lien d'invitation.

## 2. Pour les Bailleurs (Propriétaires)

Votre tableau de bord centralise la gestion de vos biens immobiliers.

### Gestion des Établissements
1.  Allez dans **Mes Établissements**.
2.  Cliquez sur **Ajouter** pour créer une nouvelle propriété.
3.  Configurez les **Chambres** et le **Mode Financier** (répartition des charges).

### Gestion des Locataires (Baux)
1.  Dans la fiche de l'établissement, sélectionnez une chambre.
2.  Cliquez sur **Ajouter un Locataire**.
3.  Remplissez les informations (Nom, Email, Date d'entrée). Le locataire recevra une notification.

### Suivi Financier
*   **Valider un Paiement** : Lorsqu'un locataire déclare avoir payé, une notification apparaît. Vérifiez la preuve (image/PDF) et cliquez sur **Valider** ou **Refuser**.
*   **Générer une Facture** : Le système crée automatiquement les loyers, mais vous pouvez ajouter des frais ponctuels (ex: régularisation).

### Maintenance
*   Consultez l'onglet **Tickets** pour voir les problèmes signalés.
*   Mettez à jour le statut (En cours, Résolu) pour informer les locataires.

## 3. Pour les Locataires (Colocataires)

Votre espace vous permet de gérer votre vie dans le logement.

### Payer son Loyer
1.  Sur le tableau de bord, consultez le montant dû dans **Mes Finances**.
2.  Effectuez le virement bancaire habituel.
3.  Cliquez sur **Déclarer un Paiement** et téléversez une capture d'écran ou le PDF de la banque.
4.  Une fois validé par le propriétaire, vous recevrez une quittance.

### Vie en Communauté (Tâches Ménagères)
1.  Consultez la section **Tâches (Chores)** pour voir votre tour de rôle.
2.  Une fois la tâche faite, prenez une photo et cliquez sur **Marquer comme Fait**.
3.  **Valider les autres** : Confirmez que vos colocataires ont bien effectué leurs tâches pour maintenir l'harmonie.

### Signaler un Problème
1.  Allez dans **Maintenance**.
2.  Cliquez sur **Nouveau Ticket**.
3.  Décrivez le problème (ex: "Fuite lavabo SDB") et ajoutez une photo.

## 4. Pour les Administrateurs

Vous avez une vue d'ensemble sur la plateforme.

### Configuration Globale
*   Accédez aux **Paramètres** pour changer le logo, les couleurs ou les textes légaux (CGU, Mentions Légales).
*   Configurez les liens vers les réseaux sociaux et le pied de page.

### Supervision
*   Le **Tableau de Bord Admin** affiche les revenus globaux (SaaS) et le nombre d'utilisateurs actifs.
*   Vous pouvez exporter les statistiques au format PDF.
